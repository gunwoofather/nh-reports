1~6회 거래 기존 고객 대상
UX / Product
하위전략 01 · 온보딩 UX 개선
  • 해외주식 주문 경로 홈 화면 전면 배치
  • 환전 → 매수 단일 화면 통합 (원화 입력)
  • 가입 온보딩 플로우 내 해외주식 체험 콘텐츠 삽입
  • 관심 종목 설정 / 테마 큐레이션 화면 제공
  • 소수점 매수 안내 팝업 및 UI 노출
하위전략 01 · 거래 직후 주식모으기 유입
  • 거래 완료 직후 주식모으기 안내 화면 자동 노출
  • "방금 매수한 종목, 자동으로 모아보세요" CTA 삽입
  • 거래 화면 → 주식모으기 신청 단계 최소화
  • 앱 내 주식모으기 진입 배너 상시 노출
  • 소액 자동 매수 시뮬레이터 UI 제공
Marketing / 이벤트
하위전략 02 · 30일 한정 인센티브 캠페인
  • 가입 후 30일 내 첫거래 고객 대상 이벤트 운영
  • 시간 제한 기반 전환 촉진 오퍼 설계
  • 해외주식 첫거래 전용 혜택 패키지 기획
  • 이벤트 랜딩 페이지 / 앱 배너 제작
하위전략 02 · 세그먼트별 가입 캠페인
  • 1~3회 구간: 첫 약정 진입 혜택 중심 캠페인
  • 4~6회 구간: 경험 후 이탈 고객 재가입 오퍼
  • 구간별 차별화 메시지 / 크리에이티브 제작
  • "매달 1만원 자동 미국주식 모으기" 소구 메시지
  • 앱 내 타겟 배너 및 팝업 노출
CRM / 알림 자동화
하위전략 03 · 맞춤 Push / 알림 자동화
  • D+7 / D+14 / D+21 개인화 종목 추천 알림 발송
  • 비전환 고객 행동 유도 메시지 자동 트리거
  • 시황 / 관심 종목 이슈 알림 자동 발송
  • 미전환 고객 단계별 넛지 시나리오 설계
하위전략 03 · 약정 지속성 관리
  • 월별 적립 현황 / 수익률 리포트 자동 알림
  • 서비스 체감 가치 강화 콘텐츠 발송
  • 약정 갱신 / 금액 증액 유도 넛지 설계
  • 장기 적립 고객 리워드 프로그램 연계
